Метрики смены

Метрики — это самая мощная часть Easy Business Space. Они позволяют владельцу настроить отчёт смены под реальный бизнес, а не под шаблон “как у всех”.

Что такое метрика?

Метрика — это дополнительная строка в отчёте смены. Она может быть:

  • ручной — сотрудник вводит значение
  • рассчитываемой — система считает по формуле

Метрики нужны, чтобы учитывать то, чего нет в стандартных полях:

  • продажи табака отдельно
  • продажи сопутствующих товаров
  • количество чеков
  • маржинальность
  • премии
  • любые показатели бизнеса

Где создаются метрики?

Путь в приложении: 

Главная → Магазины → Магазин → Отчёт за смену → Метрики

При первом входе список будет пустым — нужно добавить первую метрику.

Создание метрики

Нажмите кнопку ➕ Добавить метрику. Откроется форма:

  • Название — как будет видно в отчёте
  • Тип значения — деньги / число / количество
  • Активна — показывать в отчёте смены
  • Рассчитываемая — считать автоматически

Пример ручной метрики

Метрика: Сопутка

  • Тип: деньги
  • Рассчитываемая: ❌ нет

Теперь сотрудник каждый день вводит: 

Сопутка = 30 000₽

Это значение становится переменной для формул.

Пример рассчитываемой метрики

Метрика: Средний чек

  • Тип: число
  • Рассчитываемая: ✅ да
Формула: 
totalRevenue / checksCount
Теперь система сама будет считать показатель каждый раз.

Метрики как переменные

Все метрики автоматически становятся переменными, доступными в формулах:

  • Выручка
  • Наличные продажи
  • Безналичные продажи
  • Зарплата
  • Расходы
  • Любая добавленная метрика

Скрытие метрик от администратора

Владелец может настроить, какие метрики видны персоналу.

Например:

  • Сотрудник вводит только “Сопутка”
  • Прибыль и себестоимость скрыты
  • Формулы работают автоматически

Зачем это сделано?

Метрики позволяют владельцу построить систему учёта, которая подходит именно под его магазин:

  • минимум ручных расчётов
  • контроль остатков
  • понятная прибыль
  • гибкость без бухгалтерии

Следующий шаг — формулы.
