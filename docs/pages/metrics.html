Метрики

Метрики позволяют владельцу добавлять новые строки в отчёт смены. Это главный инструмент настройки приложения под конкретный бизнес.

Зачем нужны метрики?

  • учитывать дополнительные показатели
  • строить свои формулы прибыли
  • создавать категории продаж
  • делать себестоимость точнее

Пример

У вас есть две категории:

  • Табак (наценка 6.8%)
  • Сопутствующие товары (наценка 60%)

Вы создаёте метрику Сопутка, и сотрудник вводит её продажи в смене.

Типы метрик

  • Деньги
  • Количество
  • Число

Рассчитываемые метрики

Метрика может быть рассчитана автоматически через формулу. Например: 

Средний чек = Выручка / Количество клиентов
