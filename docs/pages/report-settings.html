Настройки отчёта (⚙)

Здесь владелец настраивает, как выглядит отчёт смены для себя и для администраторов: какие поля и метрики видны, как считается себестоимость по умолчанию.

Важно: это не «Настройки магазина». Видимость полей отчёта меняется только здесь.

Как открыть

Главная → Мои магазины → Магазин → Отчёт за смену → ⚙ (правый верхний угол) → Настройки отчёта

Иконка шестерёнки доступна только владельцу.

Что можно настроить

Режим себестоимости по умолчанию
  • Автоматически — по наценке из настроек магазина
  • Вручную — сотрудник вводит сумму
  • По формуле — из выбранной метрики (например COGS)
Скрытие полей и метрик от администратора

Владелец отмечает, какие стандартные поля и пользовательские метрики не показывать сотруднику.

Пример:

  • Сотрудник видит: выручку, наличные, безнал, расходы, метрику «Сопутка»
  • Скрыто: себестоимость, прибыль, маржинальность
  • Формулы при этом продолжают считаться на сервере/в фоне — сотрудник их не видит

Связь с метриками и формулами

  1. Создайте метрики: Отчёт за смену → Метрики
  2. Настройте формулы для рассчитываемых метрик
  3. В Настройках отчёта выберите режим COGS и скройте чувствительные строки от админа

Подробнее: разделы «Метрики» и «Формулы» в этом руководстве.
