Настройки приложения

Глобальные параметры профиля и интерфейса — не путать с настройками магазина и настройками отчёта. 

Главная → Настройки (иконка профиля / меню)

Основное

  • Профиль — имя, контакты, смена пароля
  • Валюта отображения — RUB, USD, EUR, TRY и др.; в интерфейсе суммы показываются в выбранной валюте
  • Язык — русский, английский, турецкий
  • Тема — светлая / тёмная (если доступно в версии)

Уведомления

  • Push о сменах, расхождениях, чате
  • Отдельное отключение уведомлений чата (8 ч / навсегда)

Хранилище и данные

  • Сводка по занятому месту (чат, поставки, чеки)
  • Очистка кэша изображений по категориям

О приложении

  • Версия приложения
  • Инструкция по работе — краткое руководство внутри приложения
  • Ссылки на политику конфиденциальности и поддержку

Три уровня настроек — шпаргалка
